## **리액트**

> 이번주 가장 중요한 주제: 리액트

### webpack for bundling

-   loader와 plugin의 역할이 다르다
    -   loader
    -   plugin

### DOM 구조

-   부모의 handler를 `props`를 이용해 사용할 수 있다.
    -   한 Component에서 일어난 변화가 형제 components에 적용되어야 할 때, 부모의 handler를 사용하면 부모의 state가 변경됨으로써 부모의 모든 자식 node들이 re-rendering된다.

### 상태관리

-   reducer를 따로 분리한다. 뷰를 깔끔하게 유지할 수 있다.
-   이거를 다른 module/library 에게 위탁할 수 있다.
-   대단한 상태관리를 하지 않을 거면 그냥 redux 없이 hook만 써도 무방.
